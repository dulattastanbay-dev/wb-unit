# WB.Unit — калькулятор юнит-экономики для селлеров Wildberries

Бесплатный онлайн-калькулятор прибыли для продавцов на Wildberries. Считает чистую прибыль с единицы товара, маржинальность, ROI и точку безубыточности.

## Что учитывает

- Цену закупки и планируемую цену продажи
- Комиссию маркетплейса (готовые ставки или своя)
- Логистику до склада и упаковку
- Логистику WB до покупателя **с учётом процента выкупа** и обратной логистики
- Расходы на рекламу (ДРР)
- Налоговый режим: УСН «Доходы» 6% или «Доходы − Расходы» 15%

## Технически

Один файл `index.html` без зависимостей и сборки: HTML + CSS + ванильный JavaScript. Работает офлайн, введённые данные сохраняются в localStorage.

## Запуск

Откройте `index.html` в браузере — этого достаточно.
